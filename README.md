## About the Project

**Cuctus Stand** -  It is a Multi-purpose Stand That Look Like a Regular Cuctus 
### Hold Devices:
- Headphone
- Watch
- Necklace
- Braclet
- Much more.


## Repository Structure
- `src/CAD/` — Fusion 360 project sources
- `production/` Production File
- `Render/` for Render Sources

### Recommended Print Specifications

| Specification | Default Value / Range |
| :--- | :--- |
| **Printer Nozzle** | 0.4 mm Stainless Steel |
| **Layer Height** | 0.2 mm |
| **First Layer Height** | 0.2 mm |
| **Nozzle Temperature** | 190°C – 240°C |
| **Build Plate Temperature** | 35°C – 65°C |
| **Sparse Infill Density** | 15% |
| **Max Toolhead Speed** | 500 mm/s |
| **Max Acceleration** | 10000 mm/s² |

- source: `Production`
